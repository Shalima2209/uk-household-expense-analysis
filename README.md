# uk-household-expense-analysis
Entry-level data analysis project using Excel to analyze household expenses and generate insights.

## Project Overview
This project analyzes monthly household expenses in the UK to identify spending patterns, cost drivers, and areas for potential savings.

## Tools Used
- Microsoft Excel

## Dataset
A manually created dataset covering household expenses from January 2024 to February 2026, including rent, groceries, utilities, transport, internet, mobile, entertainment, and miscellaneous costs.

## Key Analysis
- Calculated total and average monthly expenses
- Identified highest and lowest spending months
- Analyzed fixed vs variable expenses
- Examined category-wise spending trends

## Key Insights
- Average monthly expense is approximately £2,149.
- Rent is a fixed cost of £780 per month, accounting for about 35% of income.
- Variations in groceries, utilities, and transport are the main drivers of expense fluctuations.
- Seasonal patterns are visible in utility and grocery spending.

## Conclusion
This project demonstrates core data analysis skills such as data cleaning, exploratory analysis, visualization, and insight generation using Excel. It reflects real-world household budgeting and financial analysis scenarios.
